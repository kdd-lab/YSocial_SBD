Quick Reference Guide

User Types
YSocial supports three types of users with different access levels:

Admin
Full system access. Can create, edit, and delete users, experiments, and populations. Can create users with any role and change roles of non-admin users.

Researcher
Access to configuration pages and owned experiments. Can only create regular users (not admin or researcher).

User
Limited access to assigned experiments only. Cannot access admin panel.

Submission Limit
Admin users can set the maximum number of submitted experiments per account. The default is 3.
