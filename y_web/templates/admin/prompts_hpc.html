HPC Prompts Guide

HPC Prompts Structure
HPC prompts use a system/user template pattern:

System Template: Defines agent persona and context
User Template: Provides specific instructions and input

Available Variables
Prompts support template variables using curly braces:

{persona} - Agent archetype persona
{name} - Agent name
{age} - Agent age
{gender} - Agent gender
{nationality} - Agent nationality
{profession} - Agent profession
{political_leaning} - Agent's political leaning
{toxicity} - Toxicity level instruction
{oe}, {co}, {ex}, {ag}, {ne} - Big-5 personality traits
{post_content} - Content of a post
{author_name} - Post author name
{day}, {slot} - Simulation time context
{topic_instruction} - Topic guidance
{article_title}, {article_text} - News article content
{website_name} - News source name
{image_description} - Image description
{emotion_list} - Available emotions
{opinion_options} - Opinion categories

Archetype Personas
The three agent archetypes have distinct characteristics:

Validator: Skeptical, fact-checks, brief responses
Broadcaster: High energy, viral-focused, controversial
Explorer: Curious, asks questions, seeks new perspectives
